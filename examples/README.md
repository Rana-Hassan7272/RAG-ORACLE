# Examples

This directory contains example usage scripts for the RAG Failure Attribution System.

## Files

- **example_usage.py**: Basic example showing how to initialize the pipeline, run queries, and get root cause analysis
- **generate_dashboard.py**: Example script for generating system health dashboards

## Usage

Make sure you have the package installed and your environment variables configured (`.env` file with API keys).

```bash
# Basic usage example
python examples/example_usage.py

# Generate dashboard
python examples/generate_dashboard.py
```
